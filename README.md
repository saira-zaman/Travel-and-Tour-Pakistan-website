# Travel-and-Tour-Pakistan-website
 Travel & Tour Pakistan is a fully functional travel booking website designed to showcase Pakistan’s top destinations and simplify trip planning for users. The platform allows visitors to book trips and send inquiries through intuitive forms, with all data securely stored in a MySQL database.

Key Features:
Trip Booking: Collects user details, destination, travel date, and notes.
Contact Form: Enables visitors to send messages or queries directly to the admin.
Database Integration: Backend powered by PHP and MySQL for reliable data management.Responsive Design: Optimized for desktop, tablet, and mobile devices.
Technologies Used: HTML, CSS, PHP, MySQL

This project demonstrates a clean user interface, functional backend, and practical use of web technologies for real-world travel applications.
